# Hướng dẫn nhanh: Tạo Service Account cho Google Sheets API

1) Tạo project trên Google Cloud và bật Google Sheets API.
2) Tạo Service Account và tạo key JSON, tải về.
3) Chia sẻ Google Sheet với email service account (Editor).
4) Cập nhật config.yaml với spreadsheet_id và credentials_path.
5) Cài dependencies: python -m pip install -r requirements.txt
6) Thử:
   python C:\Quanlil_live\tools\run_tests.py
